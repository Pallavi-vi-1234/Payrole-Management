# 💼 Payroll Management System

## 📌 Overview
The Payroll Management System is a simple web-based project developed to manage employee salary details efficiently. It provides a basic and user-friendly interface to organize payroll data.

## 🚀 Features
- Manage employee salary details  
- Simple and easy-to-use interface  
- Organized payroll structure  

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  

## 📂 Project Structure
Payroll-Management/
│── payrole.html
│── style.css
│── script.js

## ▶️ How to Run
1. Download or clone this repository  
2. Open the project folder  
3. Open `payrole.html` in any browser
